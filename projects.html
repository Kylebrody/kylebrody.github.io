Gui Automation

Easy Utilities

A PowerShell GUI for single click diag info. Available now.

Get it on Github

I built this GUI as a way to practice making GUI forms in PowerShell, while hopefully making something with some utility. It's purpose is simple - Carry out tasks you would normally do in the shell/CMD for troubleshooting, without having to know how to use the command line. Additionally, it has single click cache clearing for Chrome and Edge. I built this thinking of people I know who may need to carry out these tasks, but don't have the know how. For this reason, the results of each test are sent to a .txt file and opened automatically. I also narrowed down the scope of the Get-ComputerInfor cmdlet by hosting only the most commonly needed info paramaters online, and pulling that data so that it is all that is displayed. As for the browser clearing, updates are planned for cookies, history, bookmarks etc. Chrome in particular stores this data in a multitude of places, so I even found use for this utility myself. I wrote a bit about that on the projects Github page. I hope you find it useful.
